Healthy Cooking Assistant Chatbot

This chatbot is designed to serve as a source of motivation to promote healthy meal choices amongst all people. It has many features and makes use of ChatGPT API (free version). 

It includes:
1.) Meal suggestion (BREAKFAST/LUNCH/DINNER/SNACKS)
2.) Includes vegan options 
3.) Counts calorie intake and compares with average calorie intake
4.) Uses free version of CHATGPT API

It is a python program and can be used on any python environment such as IDLE, VISUAL CODE STUDIO, JUPYTER NOTEBOOK, CODEPEN etc.
